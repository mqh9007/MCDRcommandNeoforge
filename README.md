# MCDRcommandNeoforge

---

**Unofficial NeoForge port of [MCDRCommandFabric](https://github.com/AnzhiZhang/MCDRCommandFabric).**

## Installation

Install this mod and then install MCDReforged plugin [Minecraft Command Register](https://github.com/Fallen-Breath/MCDReforgedPlugins/tree/master/src/minecraft_command_register).

## Version Compatibility

- Minecraft: **1.21.1**
- NeoForge: **>=21.1.228**
- MCDReforged Plugin: **2.15.7**

Based on [MCDRCommandFabric](https://github.com/AnzhiZhang/MCDRCommandFabric) by Andy Zhang and czruby.
